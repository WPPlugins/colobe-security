== Description ==

Colobe Security plugin is an easy way to use Colobe Protection Service ( https://colobe.net ) against the brute-force attacks, with this plugin the WordPress login page will be protected from the theft of passwords

== Installation ==

1. Install Colobe Security by uploading the files to your server.

2. Go to colobe.net and create a new free account, then add your site to your account.

3. Download the Colobe Library file from your account and upload it with the control panel of this plugin.

4. Finish :)
